# PRD: 프랑스어 습득 세션 (B1 → B2)

## 1. 개요

### 1.1 배경

과거 수강한 LMG어학원 프랑스어 수업의 학습 철학과 연습지 포맷을 기반으로, Claude가 제작·피드백하는 6개월 자기주도 학습 시스템을 구축한다. 원본 연습지 포맷은 Gmail에 보관된 Google Forms 응답 사본(2~7주차)에서 확인·분석 완료.

### 1.2 원본 수업의 학습 철학 (설계 원칙)

- 문법 강의, 교재, 단어장을 제공하지 않는다. 경험의 시간에 스스로를 두어 몸이 기억하게 한다.
- 학습자를 프랑스어에 던져놓고 '감'을 익히는 훈련을 반복한다. 다른 언어와의 1:1 대응은 하지 않는다.
- 짧더라도 내가 하고 싶은 말을 뱉는 연습을 한다.
- 모르는 단어가 있어도 단어를 찾지 않고 전체를 읽어 맥락을 파악한다. 어휘는 아티클 안에서 만난 것을 찾아보고 익힌다.
- 한 세션은 하나의 주제/시드를 가지되 마인드맵처럼 확장된다.
- 듣기/읽기: 맥락을 읽고 머릿속의 체계를 만든다 (관심 주제의 영상·기사 → 내용 설명).
- 말하기: 음악학원처럼 한다. 도~레~미~ 들리는 그대로 따라 한다.
- 일반 과정은 느슨하고 재밌게 관심사 위주, 시험대비 과정은 실전 문제와 말하기를 빡세게.
- DELF 실전 포맷을 차용한 과제를 배부한다 (작문-편지쓰기 등).

### 1.3 원본 연습지 포맷에서 추출한 핵심 장치 (DNA)

Gmail 첨부 원본 분석 결과:

1. **시드 + 가지치기**: 한 회차는 하나의 시드에서 출발해 확장 (예: batterie → batteuse → battre → à cœur battant → mettre 관용구).
2. **2지선다 유추 퀴즈 + 재치 있는 오답**: 정답 공개가 다음 문항으로 넘어가는 서사 장치 (예: "casse-noisette? 호두까기 인형 / 견과류 담는 통").
3. **실물 소재만 사용**: 유튜브/인스타 영상, 샹송, 그림책(T'choupi) 낭독 영상, 상품 게시물 스크린샷, 짧은 실제 기사. 교재형 지문 없음.
4. **레벨 분기**: 같은 소재에 수준별 다른 과제 (입문: 한국어로 내용 유추 / 상급: 들리는 프랑스어 받아적기).
5. **분량 자율 분기**: "더 공부할까요? / 오늘은 그만" 으로 코어와 확장 분리.
6. **스토리텔링 프레임의 출력 과제**: "연습실에서 만난 동생이 읽어달라고 합니다"(낭독 녹음), "추피가 되었다 치고 엄마에게 어제 일을 말해줍시다"(작문).
7. **메타인지 마무리**: "잘 알았던 건 어떻게 숙지했고, 몰랐던 건 왜 헷갈렸나 한국말로 정리" + 자유 피드백 칸.

## 2. 목표

### 2.1 학습 목표

- 현재 B1 수준에서 6개월 후 B2 수준 도달 (DELF B2 실전 포맷 수행 가능 상태).
- 초반 읽기/듣기 중심 → 점차 쓰기/말하기 출력과 피드백 구조로 확장.

### 2.2 시스템 목표

- 매일 풀 수 있는 학습지: 주 5세트, 회당 10~30분, 월 20세트, 총 120세트.
- 학습지는 인터랙티브 HTML 단일 파일로 구현.
- 학습지는 미리 제작하지 않는다. 주차별 학습 전략만 고정하고, 콘텐츠는 제작 시점에 확정한다.

### 2.3 비목표 (Non-goals)

- 문법 강의 콘텐츠 제작 (언어적 시드는 노출 설계 장치로만 사용, 설명으로 등장하지 않음)
- 단어장/어휘 리스트 제공
- 고정 교재 또는 사전 제작된 커리큘럼 콘텐츠
- 발음의 직접 교정 (Claude가 음성을 들을 수 없으므로 자가 녹음 비교 방식으로 대체)

## 3. 학습 설계

### 3.1 페이즈 구조 (24주)

**Phase 1 (1~6주차): 맥락 읽기의 재건 — 읽기/듣기 중심**

| 주차 | 기능 목표 | 언어적 시드 |
|---|---|---|
| 1 | 모르는 단어 3할 상태로 기사 요지 한 문장 요약 | présent / 빈도 표현 |
| 2 | 짧은 오디오(1분) 듣고 누가/무엇/어디 잡아내기 | y / en 대명사 |
| 3 | 문단별 소제목 붙이기 (구조 파악) | passé composé vs imparfait |
| 4 | dictée: 들리는 대로 받아쓰고 원문과 비교 | 관계대명사 qui/que/où |
| 5 | 의견 문장과 사실 문장 구분하기 | 의견 표현 (à mon avis, selon) |
| 6 | 1~5주 복습 + 첫 미니 진단 (읽기/듣기 세트) | 종합 |

**Phase 2 (7~12주차): 뱉기의 확장 — 쓰기 진입**

| 주차 | 기능 목표 | 언어적 시드 |
|---|---|---|
| 7 | 읽은 기사에 3문장 반응 쓰기 | parce que / donc / mais |
| 8 | 경험 이야기 5문장 (과거 시제 운용) | 과거 시제 혼용 |
| 9 | 비교하기: 한국 vs 캐나다/프랑스 | plus/moins/aussi... que |
| 10 | 가정해서 말하기 | si + imparfait, conditionnel |
| 11 | DELF B1 포맷 편지쓰기 | 편지 관용 표현 |
| 12 | 중간 점검: 쓰기 결과물 모아 피드백 세션 | 종합 |

**Phase 3 (13~18주차): 의견의 구조화 — B2 문턱**

| 주차 | 기능 목표 | 언어적 시드 |
|---|---|---|
| 13~14 | 찬반 이슈에 입장 + 근거 2개 쓰기 | pour/contre 구조 |
| 15~16 | 남의 의견 전달하기 | 간접화법, subjonctif 노출 시작 |
| 17~18 | 200자 내외 argumentation 작성 + 소리 내어 말하고 녹음 | 논증 연결 표현 |

- Phase 3부터 매주 금요일 세트는 답변을 대화에 제출하고 recast 피드백을 받는 것을 전제로 설계.

**Phase 4 (19~24주차): 시험대비반 모드**

| 주차 | 기능 목표 |
|---|---|
| 19~20 | DELF B2 compréhension écrite/orale 실전 포맷 |
| 21~22 | production écrite 실전 (250단어 essai, 공식 편지) |
| 23 | production orale 시뮬레이션 (주제 카드 → 입장 발표, 텍스트+녹음) |
| 24 | 종합 모의 세트 + 6개월 회고 |

- 이 구간은 원본 수업의 시험대비반처럼 느슨함을 버리고 실전 문제 중심으로 운영.

### 3.2 언어적 시드의 운용 원칙

시드는 학습지에 문법 설명으로 등장하지 않는다. 그 주의 Input 텍스트와 Écho 문장에 반복적으로 심어지는 노출 대상이며, 학습자에게는 "이번 주는 과거 얘기를 많이 하네" 정도로 느껴지는 것이 목표다.

## 4. 소재 정책

### 4.1 소재 풀 (경계, 스케줄 아님)

캐나다/프랑스 문화, 여성의 이야기, 최근 이슈, 지정학(국가별 이야기). 이 네 축은 주차별 로테이션이 아니라 **선택 가능한 경계**다. 제작 시점에 그날 끌리는 소재를 이 풀 안에서 고른다.

### 4.2 난이도 가드레일

- 지정학이라도 조항 분석이 아니라 이야기 단위로 접근한다 (예: "왜 캐나다와 덴마크는 섬 하나를 두고 위스키를 주고받았나").
- 텍스트의 언어 난이도는 B1+~B2-, 배경지식 없이 읽히는 소재만 선택한다.
- 소스 스펙트럼: RFI(Français facile 포함), 1jour1actu, France Info, TV5Monde 등.

### 4.3 노래와 T'choupi의 역할 (비중 상향 확정)

- **노래**: Écho 블록의 주력. 표준 패턴은 "가사 받아적기 → 비원어민 화자의 커버와 대조하며 틀린 발음 잡아내기 → 직접 따라 부르고 녹음"의 3단 구조. liaison, 묵음 등 듣기 정밀도와 발음을 동시에 훈련.
- **T'choupi 및 유사 아동 콘텐츠**: 소리 식별 훈련과 만만한 뱉기(스토리텔링 프레임의 심리적 진입장벽 완화) 담당. 언어 수준은 A1~A2이나 용도가 읽기 훈련이 아니므로 B1+에서도 유효.
- **기본 배합** (고정 규칙 아님, 제작 시점 기본값): 주 5세트 중 약 2세트는 노래/T'choupi류(소리·발음·따라하기 중심), 2~3세트는 영상/기사류(맥락 읽기·작문 중심). 페이즈가 올라갈수록 기사류 비중이 자연스럽게 증가.
- **최소 조건 (유일한 소재 제약)**: Phase 3부터 주 1세트 이상은 찬반이 갈릴 수 있는 이슈 소재. DELF B2 production 대비 목적.

## 5. 세트(학습지) 구조

한 세트 = HTML 단일 파일. 블록 흐름은 다음과 같다.

| 블록 | 내용 | 원본 장치 매핑 |
|---|---|---|
| Ouverture | 오늘의 테마 선언 한 줄 + 유추 퀴즈 1문항 | 2지선다 + 재치 오답, 정답이 시드를 여는 구조 |
| Input | 영상/기사/이미지 1~2개, 맥락 파악 질문 | "이 글은 어떤 내용일까요", 줄거리 쓰기, 받아적기 |
| — 분기 — | "더 할까요? / 오늘은 그만" | 코어(10~15분) / 확장(+10~15분) 분리 |
| Expansion | 시드에서 가지치기 (어간 묶기, 관용구, 변형 뱉기) | 같은 어간 묶기, 검색 과제형 |
| Production | 스토리텔링 프레임을 씌운 뱉기 과제 (작문 또는 녹음) | "~라고 치고 말해봅시다" 형식 |
| Clôture | 메타인지 1문항 + 자유 피드백 칸 + 답변 전체 복사 | "왜 헷갈렸나 정리", "피드백 받고 끝!" |

### 5.1 모드 토글

원본의 반 분기(말린꼬리/뻗은꼬리)를 **"베베 모드 / 언니 모드"** 토글로 재해석한다. 같은 세트를 컨디션에 따라 쉬운 과제(한국어 유추 중심)로 풀 수 있게 하는 이탈 방지 장치.

## 6. 기술 스펙 (HTML 학습지)

### 6.1 인터랙션 요소

| 기능 | 구현 |
|---|---|
| 퀴즈 분기 | 선택 시 정답 공개 + 다음 섹션 조건부 표시 |
| 듣기/따라하기 | Web Speech API (speechSynthesis, fr-FR)로 TTS 재생 |
| Dictée 채점 | 입력 텍스트 vs 원문 diff 하이라이트 |
| 녹음 | MediaRecorder API로 브라우저 내 녹음·재생 (자가 비교용, 제출 아님) |
| 답변 제출 | "답변 전체 복사" 버튼 → 그날의 모든 입력을 텍스트로 묶어 클립보드 복사 |
| 영상 | 유튜브 임베드 기본, 환경에 따라 막힐 수 있어 링크 병행 |

### 6.2 제약 사항

- Claude artifacts 환경에서 localStorage/sessionStorage 사용 불가. 답변은 파일로 저장되지 않으며, "복사 → 대화 제출"이 기록과 피드백을 겸하는 단일 경로다.
- 오디오 소재는 TTS와 유튜브 링크로 한정 (자체 오디오 파일 생성 불가).

## 7. 운영 루프

1. **제작**: 세트 제작 요청 시 먼저 `carryover.md`(학습 상태판)를 읽고, 재노출 대기 항목·반복 오류 패턴·소재 신호를 반영한다. 그 위에 해당 주차의 기능 목표와 언어적 시드를 얹어 소재를 확정하고 HTML 학습지 생성.
2. **학습**: 학습자가 브라우저에서 세트 수행 (10~30분, 분량 자율 분기).
3. **제출**: "답변 전체 복사"로 결과를 대화에 붙여넣기.
4. **피드백**: Claude가 recast 방식으로 피드백 (문법 강의가 아니라 자연스러운 문장으로 되돌려주기). 원본 수업에서 선생님이 하던 피드백 역할의 재현.
5. **기록·반영**: 피드백 직후 `log/S{NNN}.md`에 제출 원문과 피드백을 남기고 `carryover.md`를 갱신(재노출 큐 이동·추가, 오류 패턴 누적, 졸업 처리, 진도 +1)한 뒤 커밋한다. 여기서 드러난 약점이 다음 세트의 Écho/Expansion/Input으로 순환한다. **이 단계가 답변·피드백을 다음 세트로 넘기는 유일한 통로다** (세션 휘발성·연습지 무저장 때문). 상세 규격은 `docs/feedback-loop.md`.

## 9. 리스크 및 한계

- **학습량 총량**: 주 5회 × 10~30분 = 6개월 약 50~150시간. B1→B2 전환에 통상 필요한 노출 시간에 미달할 가능성이 있음 (정확한 필요 시간은 개인차 큼). 학습지를 척추로 삼되 학습지 밖 자유 노출(팟캐스트, 기사) 병행을 전제로 한다.
- **말하기 피드백 한계**: Claude가 음성을 직접 들을 수 없어 발음 교정은 자가 녹음 비교 방식. TTS 따라 말하기와 dictée까지가 학습지 내 커버 범위.
- **소재 자유화의 수렴 위험**: 스케줄 제거로 편한 콘텐츠에 수렴할 수 있음. Phase 3+ 주 1회 찬반 이슈 최소 조건으로 방어.
- **연속성**: 학습지가 상태를 저장하지 않으므로, 진도·이력은 리포지토리의 `log/`(회차 원문 아카이브)와 `carryover.md`(살아있는 상태판)에 영속화한다 (`docs/feedback-loop.md`). 대화·Claude 메모리는 보조일 뿐, 진실 원본은 리포지토리다. 단, 5단계(기록) 커밋을 빠뜨리면 이 연속성이 끊긴다.
